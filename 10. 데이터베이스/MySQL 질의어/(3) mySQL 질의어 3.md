## MYSQL 질의어 - 데이터베이스 관련
- 데이터베이스 목록 보기
  - show database;

- 데이터베이스 생성
  - create database <데이터베이스명>;

- 사용하고자 하는 데이터베이스 선택
  - use <데이터베이스명>;

- 데이터베이스 삭제
  - drop database <데이터베이스명>;

## MYSQL 질의어 - 테이블 관련
- 테이블 목록 보기
  - show tables;

- 테이블 생성
  - create tables <테이블명(필드명, 타입, 공백여부)>;

- 테이블 구조 보기
  - describe <테이블명>; 또는 desc <테이블명>;

- 테이블 삭제
  - drop tables <테이블명>;
